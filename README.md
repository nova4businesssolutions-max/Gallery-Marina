# Gallery Marina Furniture (جاليري مارينا)

A premium, production-ready React + TypeScript web application showcasing French classic furniture collections by Gallery Marina, located in Damietta, Egypt.

## 🚀 Stack & Technologies
* **Front-end:** React 18, TypeScript, Vite, React Router DOM
* **Styling:** Tailwind CSS (v4) with Cairo Typography and RTL layout configuration
* **Backend:** Supabase (Database queries, Category & Product metadata)
* **Storage:** Supabase Storage bucket for optimized WebP furniture showcases
* **Security:** bcryptjs client-side hash comparison for administrative login
* **Performance:** Advanced route code-splitting and dynamic image compression (WebP canvas utility)

## 📂 Structure
* `/react-app`: The complete Vite React source codebase.
* `/Assets`: Media resources, fonts, and assets.
* `/`: Static prototypes and landing page assets.
